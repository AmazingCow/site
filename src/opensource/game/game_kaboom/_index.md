# Game Kaboom


<!-- ####################################################################### -->
<!-- ####################################################################### -->

## Intro

```Game_Kaboom``` is a small _"quasi"_-remake of the Activision's Kaboom.   

It is **FREE SOFTWARE**, so you are **very** welcome to hack and share it.
