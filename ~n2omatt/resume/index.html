Ferramentas

  • Amazing Build
    Conjunto de ferramentas para agilizar o Content Pipeline em Jogos.
    Utiliza o ExtendScript Toolkit (Javascript) da Adobe para manipular os arquivos do Photoshop, Python e Shell Script para outras tarefas.
  • Bin2Header
    Transforma qualquer arquivo binário em um “C Header”.
    Escrito em C.
  • COWTODO
    Procura tags “TODO” no arquivos e apresenta-os de maneira estruturada.
    Escrito em Python.
  • Frame Merger
    Mescla uma imagem com outras localizadas em um diretório.
    Útil para adicionar marca d'água, logos, etc.
    Escrito em Python. utilizando Pygame para o mesclar as imagens, PyQt para prover a UI.
  • Gosh [1]
    Ferramenta para facilitar (em várias ordens de magnitude) a navegação por diretórios no terminal.
    Escrito em Python e Shell Script.
    [1] Gosh significa “go shell” (vá shell) mostrando seu propósito e um trocadilho com a expressão “gosh” do inglês.
  • Guard Checker
    Verifica e corrige os “include guards” em C/C++.
    Escrito em Python.
  • Image Converter
    Converte arquivos de imagem para outros formatos.
    Exemplo: Todos os arquivos de uma pasta que sejam JPG para PNG.
    Escrito em Python.
  • Image Organizer
    Escrutina todos os arquivos de imagem em diretórios separados por suas respectivas dimensões.
    Escrito em Python, utilizando PyGame para capturar a propriedades dos arquivos de imagem.
  • Linux USB Boot Creator
    Cria imagens de disco bootáveis no Linux. Um wrapper nos programas necessários porém adicionando uma camada de segurança para o usuário.
    Escrito em Python.
  • OSX USB Boot Creator
    Cria imagens de disco bootáveis no OSX. Mais especificamente é um “wrapper” para os programas necessários para criar o disco de boot (diskutil, hdiutil, dd).
    Escrito em python.
  • super-rename
    Renomeia arquivos e modifica os conteúdos destes, desde que eles casem com a expressões regulares de entrada.
    Escrito em Shell Script.

Bibliotecas

  • Color Helper Cpp
    Biblioteca que facilita a manipulação de componente de cor e transformações entre espaços de cores.
    Por exemplo: RGB -> CMY, HSV -> RGB, etc.
    Escrito em C++.
  • TermColor_cpp
    Adiciona funcionalidades como “Functors” e “stream manipulators” para uma utilização mais conveniente em C++.
    Biblioteca escrita em C++ inspirado no python-termcolor.
  • cowtermcolor_py
    Adiciona funcionalidades como “Functors” e “standalone functions” e output inteligente (as sequencias de escape somente são enviadas quando a saída é um terminal).
    Biblioteca escrita em Python inspirado no TermColor_cpp
  • libstdcow
    Biblioteca implementada em C com ideias de bibliotecas de outras linguagem.
    Tem o intuito de fazer C ser fácil para tarefas do dia dia sem perder o poder e flexibilidade.
  • UIKitHelpers
    Biblioteca para reduzir a quantidade de código repetitivo em Objective-C.
    Escrito em C, C++ e Objective-C.

Engines

  • MonsterFramework
    Tendo como base o Cocos2d-x v3.0, esta “micro engine” facilita várias tarefas comuns no desenvolvimento de jogos.
    O foco é facilitar ainda mais o Cocos2d-x como funções mais concisas e um API mais clara.
    Tarefas como enviar um “tweet”, e-mail ou interfacear com o dispositivo também são providas de uma maneira mais alto nível, tornando assim o código mais portável.
